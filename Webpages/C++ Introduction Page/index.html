Introduction to C++

C++ là một ngôn ngữ lập trình phổ thông được sử dụng rộng rãi trong nhiều lĩnh vực, bao gồm lập trình thi đấu

Ta có một đoạn code mẫu như sau:

        #include <iostream>
        using namespace std;
                      
        int main() {
          cout << "Hello World!";
          return 0;
        }
Variable in C++

Biến trong C++ là một khai niệm cơ bản và gán liền với mọi chương trình. Ví dụ về việc khai báo một biến số nguyên tên A:

        int a;

Ta có thể khởi gán biến A khi khai báo, cho nó một giá trị khởi đầu:

        int a = 10;

Từ đó, ta có thể thấy được cú pháp khai báo trong C++ như sau:

        kiểu_biến tên_biến;

Nói thêm về kiểu biến, trong C++ có các loại biến như:

  • Loại số nguyên: int , long long
  • Loại số thực: double , float
  • Loại ký tự: char
  • Loại xâu kí tự: string
  • ...

Dựa vào các kiểu biến trên, ta có thể khai báo các biến tùy với yêu cầu của ta.

Input/Output in C++

Để in ra màn hình một đoạn văn bản ta sẽ sử dụng cú pháp như sau

        cout << "Hello World" << endl;

Ở đây, Hello World là đoạn văn bản được in ra. Nếu ta có một đoạn văn bản khác, có thể đặt nó trong dấu "văn bản" và dùng cú pháp như trên.

Bên cạnh đó, ta có thể dùng để in ra một biểu thức hoặc một giá trị cụ thể. Ví dụ như sau


        cout << "1 + 2 = " << 1 + 2 << endl;
Với dòng code trên, nội dung sai sau sẽ được in ra màn hình: 

        1 + 2 = 3

Ta có thể hiểu rằng "1 + 2 = " là một văn bản nên khi in ra màn hình nội dung sẽ được giữ nguyên. Ngoài ra, 1 + 2 sẽ là một biểu thức và sẽ được tính thành giá trị nên khi in ra màn hình giá trị của phép tính 1 + 2 sẽ được xuất ra.

Để đọc vào trong C++, ta sử dụng cin với cú pháp như sau:


        int a;
        cin >> a;

Để đọc vào, ta bắt buộc phải có biến để lưu giá trị. Từ đó, ta có thể làm việc với các biến sau khi đọc giá trị từ đầu vào.

If-else in C++

Trong C++ tồn tại cấu trúc If-else để kiểm tra điều kiện và thực thi code nếu như điều kiện ấy thỏa mãn. Ví dụ như sau


        if(n % 2 == 0)
          cout << n << " là số chẵn" << endl;
        else 
          cout << n << " là số lẻ" << endl;

Khi đoạn code trên được thực thi, chương trình sẽ kiểm tra nếu n chia hết cho 2 thì n là số chẵn, ngược lại sẽ biết được n là số lẻ. Với cấu trúc:


        if(Điều_kiện){
          code A
        }else{
          code B
        }

Đây là dạng cấu trúc tổng quát của If-else, nếu Điều_kiện thỏa mãn thì đoạn code A sẽ được chạy, ngược lại code B sẽ được chạy.
